Este proyecto esta basado en el control de personal con el fin de tener un
control historico de personas que ingresen a hospitales, alcaldias, colegios
etc..

Este proyecto esta creado con Python y Mysql utilizando Docker y esta enfocado
en integración continua.

Para ejecutar la aplicación se debe crear la imágen de python ubicada en la
carpeta /App y después hacer un docker compose up esto hará correr el proyecto
en el puerto 3090.
